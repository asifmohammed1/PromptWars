# ⭐ StarDev — AI-Powered Code Portfolio Builder

**StarDev** is a premium, single-file developer portfolio generator. It transforms your GitHub profile into a stunning showcase in seconds. 

## 🚀 Live Demo
[View Live Site](https://prompt-wars-seven.vercel.app/)

## ✨ Key Features
- **Zero Configuration**: A fully functional Single-Page Application (SPA) in a single HTML file.
- **Dark Glassmorphism UI**: High-end modern design with backdrop blurs and CSS-only glows.
- **Tech Stack Analysis**: Visualizes your top languages based on real-time GitHub data.
- **Responsive & Accessible**: Optimized for mobile and desktop, with explicit interaction fixes.
- **Static Portability**: No build system required. Just open `index.html` and it works.

## 📁 Project Structure
- `index.html`: The entire application (Structure, Styles, and Logic).
- `README.md`: Project documentation.

## 🚀 Getting Started
Simply open `index.html` in any modern web browser or host it as a static site.
```bash
# Optional: Serve locally
npx -y serve .
```

## 🛠️ Tech Stack
- **HTML5 / CSS3**: Grid, Flexbox, Animation, Glassmorphism.
- **Vanilla JS**: GitHub REST API integration, dynamic DOM rendering.
- **Google Antigravity**: Developed using AI-native intent-driven workflows.
